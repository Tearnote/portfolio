    My projects

    Minote

    A personal experimental renderer project. Written in C++ and GLSL, using Vulkan for graphics via the vuk library. Renders high triangle counts using visibility buffer and meshlets via mesh shader emulation. The image is antialiased using a custom solution that temporally amortizes MSAA resolve.

    Sources

    Gamecraft

    A static website for a fictional videogame creation program. Plain HTML5 and CSS3, without frameworks. No Javascript, aside from an inline one-liner to trigger audio playback. Fully responsive layout, with validated and semantic markup.

    Sources, Live version

    Village of Chaos

    An incremental (idle) browser game about managing a small village, with the ultimate goal of unraveling the mystery of a distant monolith. Written in pure HTML5, CSS3 and ES6 Javascript, without libraries of frameworks. Features media queries to switch between desktop and mobile layouts, and uses a WebManifest to allow installation as an app on a mobile device.

    Sources, Live version
